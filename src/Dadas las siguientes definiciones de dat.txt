Dadas las siguientes definiciones de datos

type alias Coordinate =
    { x : Int
    , y : Int
    }

type alias TetraCoords = List Coordinate

type Tetramino = I | O | T | S | J | Z | L 

Termina esta función con la definición de las coordenadas de cada 
tipo de Tetramino en su rotación 0.

tileSvg : Tetramino -> List Coordinate
tileSvg t = 
    case t of
        I -> 
            [ {x = 0, y = 1}
            , {x = 1, y = 1}
            , {x = 2, y = 1}
            , {x = 3, y = 1}
            ]
